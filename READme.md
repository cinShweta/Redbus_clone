# 🚌 RedBus Clone

A responsive web application that replicates the core functionality of RedBus, a popular online bus ticket booking platform in India.

## About This Project

This is my first web development project built as part of my B.Tech CSE studies. It demonstrates my understanding of HTML, CSS, and modern web design principles including layouts, forms, cards, and responsive design.

## ✨ Features

- **Navigation Bar** - Easy access to Home, Bus Tickets, Offers, Contact, and Login pages
- **Search Functionality** - Search for buses by origin, destination, and date
- **Popular Routes** - Display of popular bus routes with starting prices
- **Special Offers** - Current discount codes and bank offers
- **Why Choose Us Section** - Features highlighting the platform benefits (2000+ buses, secure payment, etc.)
- **Responsive Design** - Beautiful UI with custom styling and interactive buttons
